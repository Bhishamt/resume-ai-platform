# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security related issues, please email `security@example.com` instead of using the issue tracker.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you're curious about the status of your report, feel free to reply to the email.
